# Flight-Delay-Prediction-using-ML
## Overview
This project predicts airline flight delays using machine learning models trained on the Airline Delay Dataset. The goal is to classify flights as delayed or on time using route, airline, and delay statistics.

## Project Contents
- `Flight_Delay_Prediction.ipynb` — Jupyter notebook containing data preparation, model training, evaluation, and model saving.
- `Airline_Delay_Cause.csv` — Dataset used for training and evaluation.
- `flight_delay_model.pkl` — Saved trained Random Forest model.
- `Month 5 - Project Title Flight Delay Prediction Using Machine Learning.pdf` — Project brief and requirements.

## Features
- Data cleaning and preprocessing
- Categorical encoding for airline and airport features
- Feature engineering for date fields
- Model training with Logistic Regression and Random Forest
- Evaluation using accuracy, precision, recall, confusion matrix, ROC curve, and classification report
- Model serialization to `.pkl`
